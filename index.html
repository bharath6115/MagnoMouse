Welcome to MagnoMouse!

In this game, your mouse behaves like a Magnet! You have to determine the location of buttons which exhibit metallic property. Hover over the button, you will be alerted if the button is metallic! Your goal is to report all the metallic buttons, you can quit midway if it is too difficult.

Note: Each button has 0.1% chance to be metallic!
Select Difficulty:
Easy: (max 5 attempts)
Buttons have response time of 0ms.
Medium: (max 5 attempts)
Buttons have a minimum response time of 1000ms, maximum being 2000ms.
Hard: (max 3 attempts)
Same as medium difficulty but you must finish the game within a set time.

Hover over each button, if a button shows metallic nature then the grey display above will turn green. Then click that button to select it. If the button is not metallic, then upon clicking it, it will turn red and you will LOSE a heart. Else, it will turn green. Identify all the metallic buttons before your hearts run out to win!

Note: Each button has 0.1% chance to be metallic!
